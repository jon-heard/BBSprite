Detailed Description

The MonoBehavior responsible for determining the animations of a BbSprite.

BbSpriteAnimation holds multiple BbSpriteAnimation objects, with one of them being active at any given time. The activated animation is advanced each update and its current frame is available from the CurrentFrame property. This is used by the BbSprite behavior to determine which cell to draw at any given time.
