Detailed Description

This layout allows for a customized layout based on breaking an atlas into a contiguous grid of cells.

It also expects that they are ordered by rotation of the BbSprite within the cell. For more complex atlas layouts, see the Shuffle class (available soon).

int [] Assets.BbSprite.layouts.Grid.DirectionOffsetList = { 0, 2, 3, 1 }

The list of frame offsets for viewing angles in clockwise order.

The full rotation is broken up into segments assigned to each offset starting at 0 degrees being at the center of the first offset. Example: "0,2,3,1" maps these angle ranges to frame offsets:

  • 0-45 degrees = 0 frame offset
  • 45-135 degrees = 2 frame offset
  • 135-225 degrees = 3 frame offset
  • 225-315 degrees = 1 frame offset
  • 315-360 degrees = 0 frame offset

bool Assets.BbSprite.layouts.Grid.VerticalIndexing = true

Determines how the cells are counted: column first or row first.

True - Cell indices are ordered by column first, then row.
False - Cell indices are ordered by row first, then column. (like reading).
True is detault as, while it is less intuitive, it makes it much easier to add new cells without needing to change frames of animations.
